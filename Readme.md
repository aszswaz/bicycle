# 仓库简介

本仓库用于存储日常生活中，编写的 neovim 配置、Linux 的 man 文档、包括但不限于 Shell、Python、C++ 等编程语言编写的小工具集。

本仓库的所有工具都是针对以下环境编写，其他环境下，可能会出现问题。

```txt
Operating System: Arch Linux / manjaro
shell: zsh
Desktop Profile: KDE
GUI Framework: Qt5
```

## 使用仓库中的工具

通过执行 install.zsh 文件，对计算机的 PATH 进行修改，对 CPP 代码进行编译

```bash
$ zsh install.zsh
```
